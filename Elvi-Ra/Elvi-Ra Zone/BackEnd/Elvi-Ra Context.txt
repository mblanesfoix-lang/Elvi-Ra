Elvi-Ra Flujo de Trabajo

Flujo de trabajo 1: Context Fuel

Elvi-Ra es la mano derecha de South Navarre Fresh Innovations Corporation (S-NFI Corp.), se le añadira contexto, información, métricas... sobre S-NFI, llamado " Data Base Elvi-Ra".

Flujo de trabajo 2 : Carpeta "Mesa"
La carpeta llamada "Mesa" ubicada en "Elvi-Ra" estará dirigida a los prompts/instrucciones de cada panes de trabajo/"agente". Actualizada, es decir cada vez que se modifique algo cada archivo de "agente" se modificara para que este al dia .

Flujo de Trabajo 3: Data Base
Elvi-Ra será la encargada de alimentar cada "agente" con la información necesaria ar funcionar correctamente.




Elvi- Ra no es una simple plataforma o IA, es una fuente de datos para dirigir los "agentes" creados .

- Elvi-Ra creara y monitorizara un Dashboard donde se ubicara cada "Agente", el T'Controler (Tokens controler) y las credenciales y usuarios.
- Base de datos con la cual se alimenta a los "Agentes"
- Controlador e Tokens (T'COntroler)
-Monitorizacion de usuarios


Login 1a Fase:
Cada persona dispone de X acceso a la plataforma, las credenciales están ubicadas en "Elvi-Ra"-"Elvi-Ra Zone"-"Creedenciales"
